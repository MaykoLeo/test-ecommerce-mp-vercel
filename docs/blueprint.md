# **App Name**: ScentPay

## Core Features:

- Product Catalog: Display a list of available perfumes with details like name, brand, description, and price.
- Product Detail Page: Show detailed information about a selected perfume, including high-quality images and customer reviews.
- Shopping Cart: Allow users to add perfumes to a shopping cart and adjust quantities.
- Checkout Process: Guide users through a simplified checkout process to input shipping information and select payment via Mercado Pago.
- Mercado Pago Integration: Integrate Mercado Pago Pro to process payments securely.
- Order Confirmation: Display an order confirmation page upon successful payment, with order details and tracking information.
- Order Management: Persist order details such as the customer's information, order date, purchased items and price in the database

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of luxury and calmness.
- Background color: Light grey (#F0F0F0), nearly white, providing a clean backdrop.
- Accent color: Pale violet-red (#D87093) for calls to action and highlights, offering a touch of sophistication.
- Body and headline font: 'Alegreya', a humanist serif, to convey elegance and intellectual appeal.
- Use minimalist line icons for navigation and product categories, ensuring clarity and a modern feel.
- Employ a clean and spacious layout with clear visual hierarchy to highlight products and ensure easy navigation.
- Subtle transitions and hover effects to enhance user interaction without being distracting.
